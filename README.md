# Sistem PKL Prodi Sains Data UPN Veteran Jatim

Proyek ini akan digunakan untuk mengelola proses Praktik Kerja Lapangan (PKL)
mahasiswa Prodi Sains Data: pendaftaran, penempatan, logbook, bimbingan, dan penilaian.

Struktur awal:
- `backend/` — kode backend (Django / API)
- `docs/` — dokumen analisis (use case, ERD, dsb.)
